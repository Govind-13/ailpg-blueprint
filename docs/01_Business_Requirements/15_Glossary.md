# 1. Purpose

The glossary provides consistent definitions for all project stakeholders, including business users, developers, testers, architects, and administrators.

---

# 2. Business Terms

| Term | Definition |
|------|------------|
| Course | A complete learning module created from one or more MP4 videos. |
| Lesson | A single learning unit within a course. |
| Interactive Lesson | A lesson containing quizzes, checkpoints, subtitles, and interactive elements. |
| Teacher | User responsible for creating and publishing learning content. |
| Student | User consuming learning content. |
| Reviewer | User responsible for validating AI-generated educational content. |
| Institution | A school, college, coaching center, or organization using the platform. |
| Organization | A tenant or customer account within the platform. |
| Subscription Plan | Defines available features and usage limits. |

---

# 3. AI Terms

| Term | Definition |
|------|------------|
| AI Job | A background process that converts an uploaded MP4 into learning assets. |
| Speech-to-Text (STT) | Conversion of spoken audio into text. |
| OCR | Optical Character Recognition used to extract text from video frames. |
| Formula Recognition | Detection and conversion of mathematical expressions into structured formats such as LaTeX. |
| Translation | AI-assisted conversion of educational content into another language. |
| LLM | Large Language Model used for language understanding and content generation. |
| Confidence Score | Numerical estimate of AI prediction confidence. |
| Regeneration | Re-running one or more AI processing stages after review. |

---

# 4. Technical Terms

| Term | Definition |
|------|------------|
| API | Application Programming Interface used for communication between services. |
| Backend | Server-side application logic and services. |
| Frontend | User-facing web interface. |
| Dashboard | A visual interface displaying metrics, actions, and reports. |
| HTML Lesson | The generated interactive lesson delivered in a web browser. |
| Object Storage | Storage service for videos, images, and generated files. |
| CDN | Content Delivery Network used to distribute static assets efficiently. |
| Metadata | Information describing uploaded content, such as duration and language. |

---

# 5. Security Terms

| Term | Definition |
|------|------------|
| Authentication | Verifying a user's identity. |
| Authorization | Determining what a user is allowed to access. |
| RBAC | Role-Based Access Control used to manage permissions. |
| MFA | Multi-Factor Authentication for enhanced account security. |
| Audit Log | Immutable record of important user and system actions. |

---

# 6. Analytics Terms

| Term | Definition |
|------|------------|
| KPI | Key Performance Indicator used to measure success. |
| Event | A tracked user or system action, such as video playback or quiz completion. |
| Heatmap | Visual representation of frequently viewed or replayed video sections. |
| Completion Rate | Percentage of learners who finish a lesson or course. |
| Engagement | Measurement of learner interaction with educational content. |

---

# 7. Streaming Terms

| Term | Definition |
|------|------------|
| Adaptive Streaming | Dynamically adjusts video quality based on network conditions. |
| Playback Speed | Rate at which the video is played (e.g., 0.5x, 1x, 2x). |
| Subtitle | Text synchronized with spoken content. |
| Checkpoint | A predefined timestamp where an interactive activity or quiz appears. |
| Resume Playback | Continue watching from the last saved position. |
